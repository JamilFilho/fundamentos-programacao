## Fases para o desenvolvimento de um programa

O desenvolvimento de um programa deve passar pelas seguintes fases:

1. Planejamento: os cronogramas, objetivos, missão.
2. Análise:
3. Projeto:
4. Implementação
5. Testes:
6. **Manutenção:** ajustes, correções ou atualizações.

A lógica de programação estabelece os métodos/estrutura de pensamento para se desenvolver um programa, indiferente da linguagem utilizada. É a base utilizada para resolver um determinado problema.

## Algoritmo

O algoritmo é a instrução dada para que determinada ação seja realizada pelo processador computacional. Um algoritmo precisa:

1. Ter objetividade.
2. Executar uma ação por instrução (definição de responsabilidades).
3. Número finito de etapas.
4. Conjunto de dados/instruções de entrada definidos.
5. Uma ou mais saídas/resultados.

Um algoritmo se fundamenta em três estruturas básicas:

1. Sequência
2. Seleção
3. Repetição

Todo algoritmo trabalha seguindo uma sequência lógica de execução: entrada → processamento → saída

- **Entrada:** dados ou instruções iniciais.
- **Processamento:** execução das instruções.
- **Saída:** resultado final.

Exemplo de representação de algoritmo para estudo de lógica de programação:

```pseudocode

ALGORITMO "SOMA" // nome do algoritmo

VAR
    N1, N2, SOMA // variáveis

INICIO // início do bloco de instruções
    Leia N1;
    Leia N2;
    SOMA <- N1 + N2;
    Escreva SOMA
FIM
```
