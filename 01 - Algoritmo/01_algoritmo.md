A lógica de progração estabelece os métodos para se desenolver um programa, indiferente da linguagem utilizada. É a base para se resolver um determinado problema.

## Algoritmo

O algoritmo é a instrução dada para que determinada ação seja realizada.

Um algoritmo precisa:

1. Ter objetividade
2. Executar uma ação por instrução (definição de responsabilidades)
3. Número finito de etapas
4. Conjunto de dados/instruções de entrada definidos
5. Uma ou mais saídas/resultados

Um algoritmo se fundamenta em três estruturas básicas:

1. Sequência
2. Seleção
3. Repetição

Todo algoritmo trabalha seguindo uma sequência lógica de execução: entrada -> processamento -> saída

- Entrada: dados ou instruções iniciais
- Processamento: execução das instruções
- Saída: resultado final

Exemplo de representação de algoritmo para estudo de lógica de programação

```pseudocode

ALGORITMO "SOMA" // nome do algoritmo

VAR
    N1, N2, SOMA // variáveis

INICIO // início do bloco de instruções
    Leia N1;
    Leia N2;
    SOMA <- N1 + N2;
    Escreva SOMA
FIM
```
